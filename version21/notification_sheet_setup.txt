Google Sheet tab name: Notices

Required headers in row 1:
title | text | priority | link

How it works:
1. Create a new Google Sheet tab named Notices.
2. Copy the headers exactly: title, text, priority, link
3. Add one notice per row.
4. priority must be a number from 1 to 5.
5. link is optional.
6. Re-deploy your Apps Script after updating Code.gs so the app can read the new Notices sheet.

In the app:
- Bell icon shows the notice list overlay.
- Overlay list shows only title and priority.
- Tapping a notice opens the full text and optional link.
